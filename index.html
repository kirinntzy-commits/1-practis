Менің тәжірибем

Қандай құрал ең көп көмектесті?

Осы жұмысты орындауда маған Claude ең көп көмектесті. Оның HTML, CSS және JavaScript туралы түсіндірмелері өте анық және деталды болды. Мысалы, Flexbox пен CSS Grid айырмашылығын нақты мысалдармен түсіндіріп берді. GitHub Copilot IDE-де код жазғанда өте ыңғайлы — функция атауын жазсаң, толық реализацияны ұсынады.

ЖИ нені дұрыс немесе ыңғайсыз жасады?

ЖИ HTML структурасын жасауда өте жақсы жұмыс жасады, бірақ кейде тым күрделі немесе контекске сай емес код ұсынды. Мысалы, ChatGPT кейде React-тегі компонент ретінде жазып берді, ал мен таза JavaScript қолдануым керек болды. Мұны нақтырақ prompt жазу арқылы түзеттім: «тек vanilla JS, ешқандай фреймворксіз» деп нақты айттым.

ЖИ 5 жылдан кейін әзірлеушілерді қалай өзгертеді?

Менің ойымша, 5 жылдан кейін ЖИ рутиналық код жазуды толық автоматтандырады. Әзірлеушілер синтакс жаттаудан гөрі архитектура жасауға, бизнес логикасын шешуге және ЖИ шығарған кодты тексеруге уақыт жұмсайды. «Prompt engineering» жаңа маңызды дағды болады. Дегенмен, негізгі алгоритмдер мен деректер құрылымдарын терең түсінетін мамандар әрдайым бағалы болады — себебі ЖИ қателерін тек тереңдік білім арқылы анықтауға болады.
